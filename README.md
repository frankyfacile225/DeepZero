Your computer needs specific parts to run DeepZero well. Check these specs before you start:

- Processor: A modern multi-core CPU with at least 4 cores.
- Memory: 16 GB of RAM or more.
- Storage: 10 GB of free space on a solid-state drive.
- Network: A stable internet connection for processing tasks.

## 🚀 Getting Started

Once you install the software, you can begin your first check. DeepZero uses AI agents to look at your driver files. 

1. Launch the DeepZero application from your Windows Start menu.
2. Accept the terms of use when the window opens.
3. Choose the folder containing your Windows drivers. Most users check files in System32.
4. Click the Start Analysis button.
5. Let the software run in the background. It will process thousands of files automatically.

## 💡 What DeepZero Does

DeepZero simplifies security research. It performs several steps for you:

- Parsing: The program reads the technical data inside each driver.
- Decompiling: It converts machine code into a format that the AI can read.
- Analyzing: The program looks for dangerous patterns in the input and output control codes.
- Reporting: It saves a list of every potential risk it finds.

## 🛠️ Frequently Asked Questions

### Does this program harm my computer?
No. DeepZero reads files. It does not change or delete your system data. It only observes your drivers to find potential issues.

### How long does an analysis take?
The time depends on the size of your driver folder. A standard check usually takes about one hour. Larger folders might take longer.

### Can I run other programs while DeepZero works?
Yes. DeepZero works in the background so it will not stop you from using your computer. You might notice your computer runs slower if you perform heavy tasks like video editing at the same time.

### Where can I see the results?
The program creates a folder on your desktop called DeepZero-Reports. You will find text files inside that explain any vulnerabilities found during the scan.

### Do I need to update DeepZero?
Updates keep the AI models accurate. Check the website for new versions each month. If the program shows a notification about an update, click it to download the latest features.

## 🛡️ Privacy and Safety

DeepZero processes your files locally on your computer. Your driver data does not leave your machine unless you share the report files with someone else. This protects your system information from outside parties. Keep your software updated to ensure the security models remain effective against new threats.

## 📈 Understanding the Reports

The report files provide simple text summaries. Each entry includes the path to the driver and a description of the risk. You do not need technical skills to read the findings. The software highlights high-risk items at the top of the list. If you see a high-risk item, follow the link provided in the report to learn how to contact the driver developer. You can fix most issues by checking for updates from your hardware manufacturer.

## 🔧 Managing Program Settings

You can adjust how DeepZero works via the settings menu. 

- Scan Priority: Change how much power the program uses. Use Low priority if you want to use your computer while it scans. Use High priority to finish the scan faster when you step away from your desk.
- File Types: By default, DeepZero checks system drivers. You can add extra file formats if you want to scan other software components.
- Automatic Cleanup: This feature deletes old report files after 30 days to save space on your hard drive.
